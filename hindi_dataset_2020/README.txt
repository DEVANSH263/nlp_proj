HASOC 2020 — Hindi Dataset
============================

Download from: https://hasocfire.github.io/hasoc/2020/dataset.html

Expected files in this folder:
  hasoc2020_hi_train.tsv    — training split
  hasoc2020_hi_test.tsv     — test split (gold labels)

TSV column format (same as HASOC 2019):
  text_id  |  text  |  task_1  |  task_2  |  task_3

task_1 values: HOF (Hate and Offensive), NOT (Non-offensive)
Text language: Hindi (Devanagari script)
